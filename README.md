# Wallet Authorization Tool

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/soco1988/testanything.git
   cd testanything
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure the environment variables:
   Create a `.env` file in the root directory and add the necessary variables:
   ```
   WALLET_ADDRESS=your_wallet_address
   API_KEY=your_api_key
   ```

## Usage
Run the authorization tool:
```bash
node walletAuth.js
```

Follow the prompts to authorize your wallet. Ensure that you have a valid wallet address set up in the `.env` file.

## Troubleshooting
- Ensure all dependencies are installed correctly.
- Check the `.env` file for correct wallet address and API key.
